networx-graph
http://spenibus.net
https://github.com/spenibus/networx-graph-php-cli
https://gitlab.com/spenibus/networx-graph-php-cli


Creates a visual representation of an exported csv from networx.
The csv file must be exported from the "Per Day" tab with "volume unit" set to "bytes" rather than "automatic".

Networx: https://www.softperfect.com/products/networx/

Sample csv file and associated graph available in "./sample".
